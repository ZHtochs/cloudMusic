# cloudMusic

一个网易云音乐歌单的爬虫，利用多线程实现了爬虫操作，并用的springboot做了个简单的界面。

歌曲评论数的爬取可以使用这个接口http://music.163.com/api/v1/resource/comments/R_SO_4_+（歌曲ID）

参考知乎回答
[如何爬网易云音乐的评论数？ - 肖飞的回答 - 知乎](https://www.zhihu.com/question/36081767/answer/310726622)
